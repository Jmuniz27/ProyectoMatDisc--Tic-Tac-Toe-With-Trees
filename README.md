# Instalación

```powershell
# crea entorno
python -m venv env
# en microsoft powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
# activar entorno
.\env\Scripts\activate
# instalar dependencias
pip install -r requirements.txt
# desactivar
deactivate
```
